CodeDiffNG: Advanced Source Code Diffing

CodeDiffNG is an applied PhD project being carried out at Université catholique de Louvain and Raincode Labs in 2020–2023. We are interested in researching and developing methods of differencing source code beyond the state of the art in clone detection, program differencing and model comparison, and covering industrially relevant scenarios of producing and visualising codebase progress over years of active development, or comparing different preprocessors, or even distinguishing generated code from hand-written code...
